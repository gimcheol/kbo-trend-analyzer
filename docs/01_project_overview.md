# KBO Trend Analyzer

## 1. 프로젝트 개요

KBO 선수들의 시즌 기록과 최근 경기 데이터를 기반으로
가공된 세부 스탯과 추세 분석 정보를 제공하는 RESTful API 서버입니다.

단순 누적 기록 조회를 넘어,
데이터를 계산하고 해석하여 제공하는 것을 목표로 합니다.

---

## 2. 문제 정의

현재 KBO 기록은 누적 수치 중심으로 제공되며,
다음과 같은 한계가 있습니다:

- 최근 경기 폼을 직관적으로 확인하기 어렵다
- OPS, ERA 등의 세부 지표를 직접 계산해야 한다
- 리그 평균 대비 성과를 한눈에 보기 어렵다

본 프로젝트는 이러한 문제를 해결하기 위해
기록 데이터를 가공하여 의미 있는 정보로 변환합니다.

---

## 3. 타겟 사용자

- KBO에 관심 있는 일반 팬
- 선수 성적 비교를 원하는 사용자
- 데이터 기반 스포츠 분석에 관심 있는 사용자

---

## 4. MVP 기능 범위

- 선수 목록 조회 API
- 특정 선수 시즌 기록 조회
- 타자 OPS 계산
- 투수 ERA 계산
- 최근 7경기 추세 분석
- HOT / COLD 상태 판별

---

## 5. 향후 확장 가능성

- 리그 평균 대비 지표 (OPS+ 등)
- 팀 전력 분석 기능
- 간단한 경기 결과 예측 로직
- 캐싱 및 성능 최적화
